# Java Guessing Game 🎮

This is a Java console-based guessing game that I created to practice programming and improve my skills.

## Features
- Difficulty levels (Easy, Medium, Hard)
- Points system with speed bonus
- Top 3 highscores with names
- Personal best score tracking
- Input validation
- Replay menu (continue, score, my, exit)
- Highscores saved in a file

## How to play
- Guess the number within limited attempts
- Get more points for fewer attempts and faster guesses
- Try to reach the top 3 highscores

- Java
